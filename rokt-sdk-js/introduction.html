Introduction

Rokt is a marketing technology platform that enables brands to be smarter in the transaction by optimizing the user experience and driving short and long term value. This site contains developer resources for getting started using Rokt, technical documentation, and code samples. Specific resources include documentation on Rokt’s SDKs - both web and native mobile, and the Rokt HTTP API integrations.
