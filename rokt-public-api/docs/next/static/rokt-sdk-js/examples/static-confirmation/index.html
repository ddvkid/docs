Confirmation Page Integration

Partner integration on confirmation page

This scenario involves integrating with the Rokt Web SDK in order to render placements and offers natively on your site.

Ensure that you populate the code with relevant data attributes to maximize the value Rokt can provide. For this scenario, raw email is highly recommended at a minimum, but to get the most out of Rokt optimization algorithms we suggest passing all core and transactional/vertical attributes.

Render a Placement Embedded into your page

This scenario involves rendering a placement within the content of your page. The Web SDK will need to know where on the page to render the Embedded Placement. There are two options available to achieve this:

  1. Place a new div element on your page in the location you wish to render the placement, for example: <div id="rokt-placeholder"></div>
  2. Use an existing element on your page to which the Rokt placement can be appended to.
