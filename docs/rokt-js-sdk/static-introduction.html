Introduction

The Web SDK is an integral part of the Rokt solution. With one simple, secure integration we can optimize for the broad range of objectives our clients have. For Rokt Partners, the Web SDK empowers our clients to optimize a front-end user experience through commerce optimizations and third party offers. For Rokt brands, the Web SDK closes the loop on conversion tracking, enabling Rokt to better optimize campaigns towards users that are more likely to convert on your site.
